# Anion storing, oxygen vacancy incorporated perovskite oxide composites for high-performance aqueous dual ion hybrid supercapacitors

* Publication: Journal of Energy Chemistry
  
* DOI: [10.1016/j.jechem.2024.02.032](https://doi.org/10.1016/j.jechem.2024.02.032)

This repository includes all the input files and some example output files for VASP simulation, computational structural models of LaMnO3, and the organized result.

This repository was created by Yu Chen.
